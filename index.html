Around the Web

Graph GUI

A Java graphical user interface to build and perform algorithms on the graph data structure. You can add and remove edges by clicking the screen, and can highlight connected components and cut vertices. Source code: https://github.com/danielrand/Graph-GUI

Homework Grader

As a teaching assistant for Algorithms I am tasked with grading coding projects submitted via email. There is usually around one project per week, written in either Java or C++. It became clear rather quickly that it would be impossible to manually sort through student submissions in my email and to download, run, and grade them, so I took it upon myself to automate this process. The script is a command line tool that downloads all attachments from your recent emails that conform to a search string, organizes the files neatly in directories, runs the Java or C++ code and writes output files and reports for every student in an output folder. Check it out on GitHub: https://github.com/danielrand/TA-Homework-Grader

Android Reminder App

Computer Vision

As a student I implemented a variety of computer vision algorithms such as Mathematical Morphology, Chain Code Edge Detection Method, Medial Axis Derivation/Skeleonization , Connection Components , Noise Filtering. The example above is of the input an output to my Morphology program. The input image (left) is a binary image with "meatball" objects and "spaghetti" as noise, but by using the Opening morphological operation on the input using a structured element that matches the structure of the meatballs I was able to extract just them. You can view the code to that project or any of the others by clicking on their links.

Maccabeam - A search / web crawler application

I developed this web app with the help of a couple friends. We Utilized an Agile development approach, Docker containers, a mySql Database, and deployed to Heroku. The app is capable of searching for words by either web crawling, using the Google custom search API, or by running a custom search algorithm on user uploaded JSON, CSV, or XML files. Additionally, the user can choose to select and download the results from any of the searches in the form of the their choice of the three aforementioned file formats.
Website: https://maccabeam.herokuapp.com/
Code: https://github.com/danielrand/Search-and-Reporting-Application
